The table component has all the following classes. Even `.table-striped`!

 `.table`, `.table-striped`,`.table-bordered`, `.thead-light`, `.thead-dark`, `.table-{theme-color}`, `.table-dark`

```html
<table class="table">
  <thead class="thead-light">
    <tr>
      <th>#</th>
      <th>First Name</th>
      <th>Last Name</th>
      <th>Username</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th>1</th>
      <td>Stuart</td>
      <td>Yam</td>
      <td>@StuYam</td>
    </tr>
    <tr>
      <th>2</th>
      <td>Who</td>
      <td>You</td>
      <td>@?!</td>
    </tr>
  </tbody>
</table>
```
